## Requirements
 * PHP >= 8.1
 * ext-zip - for zip and unzip functions
 * Laravel 9 or higher
 * GD Library or Imagick for [intervention/image-laravel](https://github.com/Intervention/image-laravel)
 * Bootstrap 5 and Bootstrap Icons v1.8.0 and higher
